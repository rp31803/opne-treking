🖥️ How to Run Locally

  1. Install Python 3.8+ from python.org
  2. Clone the repository:
    git clone https://github.com/rp31803/opne-treking.git
  3. Install dependencies:
    cd osint-tracker && pip install -r requirements.txt
  4. Run the application:
    streamlit run main.py
  5. Open your browser to http://localhost:8501

🎯 Features

📸 Image Analysis

Extract EXIF metadata, GPS coordinates, and timestamps from images

📄 PDF Analysis

Analyze PDF documents for text content, metadata, and intelligence

📱 Phone Tracker

Validate phone numbers and extract carrier information globally

🔍 Authenticity Check

Detect image manipulation and PDF tampering using advanced algorithms

📋 How to Deploy

  1. Fork the repository on GitHub
  2. Go to share.streamlit.io
  3. Connect your GitHub and select this repository
  4. Choose main.py as the main file
  5. Click Deploy! ✨
