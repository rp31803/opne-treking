About OSINT Toolkit

The OSINT Toolkit is a comprehensive suite of open-source intelligence tools designed for digital investigators, researchers, and security professionals. Our mission is to provide accessible, powerful tools for ethical digital investigations.

Built with Python and modern web technologies, our toolkit offers advanced capabilities for image forensics, phone number analysis, document intelligence, and location-based investigations.

We believe in transparency, security, and responsible use of technology. All our tools are open source and designed to protect user privacy while enabling thorough investigations.

Available Tools

📸

Image EXIF Analyzer

Extract GPS coordinates, timestamps, camera information, and metadata from digital images. Supports JPG, PNG, TIFF, and BMP formats.

📱

Phone Number Validator

Validate phone numbers worldwide, identify carriers, regions, timezones, and line types using comprehensive phone number databases.

📄

PDF Document Analyzer

Extract text content, metadata, embedded images, and automatically detect emails, URLs, and phone numbers within PDF documents.

🔍

Image Authenticity Checker

Use Error Level Analysis (ELA) to detect image manipulation, compression artifacts, and potential tampering.

🗺️

GPS Location Mapper

Convert GPS coordinates to human-readable addresses and display interactive maps for location-based intelligence.

🔒

Document Integrity Scanner

Check PDF files for digital signatures, metadata consistency, and potential security vulnerabilities.
